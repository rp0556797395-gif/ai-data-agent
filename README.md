🤖 AI Data Agent - Intelligent Data Manager

An interactive Streamlit application powered by Google Gemini AI, designed to analyze, clean, and merge data files intelligently.

🌟 Key Features

File Support: Seamlessly upload and process both CSV and Excel files.

AI Data Insights: Generate automatic charts and natural language summaries (Hebrew/English) based on your data.

AI-Powered Smart Delete: Clean your data using natural language commands (e.g., "Delete rows with inactive customers").

Smart Data Merging: Connect two different files with an intelligent column-merging logic that prevents duplicates and fills missing values.

Full Data Preview: View the entire dataset in a responsive, interactive table.

🚀 Getting Started

Prerequisites

Python 3.8 or higher

A Google Gemini API Key

Installation

Clone the repository:

git clone [https://github.com/your-username/ai-data-agent.git](https://github.com/your-username/ai-data-agent.git)
cd ai-data-agent


Install dependencies:

pip install -r requirements.txt


Run the application:

streamlit run app.py
